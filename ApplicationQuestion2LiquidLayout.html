The basic idea of a liquid layout is that all the information contained inside a webpage will be visible despite the size of the users window. Instead of the webpage having a set frame(no matter what, elements and words always stay in the same place), a liquid layout uses percentages of the current browsers window size to maintain the content within the window size. In other words, elements and words can adjust and don't stay in the same place(or their place largely depends on the window size, screen resolution, etc). Liquid layouts are especially useful today since mobile computing has become increasingly popular. Smart phones, tablets, laptops, desktops, and game consoles are just a few reasons why using a liquid layout is important. All these different devices potentially have a different window size. By using a liquid layout, you can be reassured that all your content will display correctly in each of these devices.

PLEASE NOTE: The buttons that cause the resizing or new window to open were implemented in JavaScript. If you are using the Chrome or Opera browser, please use the buttons in the blue column to open a new window. This was done because Chrome and Opera no longer support the window.resizeto() function. Everyone else can use the buttons in the green column to resize their window to the appropriate resolutions. Alternatively, you can just adjust the window manually by clicking on the corner or side of your window and drag it anywhere to see the liquid layout in action.
